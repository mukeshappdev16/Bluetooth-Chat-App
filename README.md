# Bluetooth Chat App

An Android application that allows users to chat with each other over Bluetooth.

## Features

- Scan for nearby Bluetooth devices.
- Pair and connect with devices.
- Real-time text messaging.

## Tech Stack

- **Language:** [Kotlin](https://kotlinlang.org/)
- **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Minimum SDK:** 24 (Android 7.0)

## Getting Started

1. Clone the repository.
2. Open the project in Android Studio.
3. Build and run the app on two physical Android devices (Bluetooth doesn't work well on emulators).
